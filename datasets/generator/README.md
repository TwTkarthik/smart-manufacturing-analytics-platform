# SMAP Synthetic Dataset Generator

**Version:** 1.0.0
**Sprint:** 3 — Database Initialization & Synthetic Data
**Python:** >= 3.11

---

## Overview

This generator produces production-realistic synthetic datasets for the SMAP
(Smart Manufacturing Analytics Platform) operational database. All data is
deterministic, referentially consistent, and configurable without code changes.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Generate all entities with default config
python main.py

# Generate with custom seed
python main.py --seed 12345

# Generate specific entities only
python main.py --entities machines,employees,production_orders

# Dry run (no file output)
python main.py --dry-run

# Parquet output
python main.py --format parquet
```

## Configuration

Edit `config/generator_config.yaml` to control:

| Section | Purpose |
|---|---|
| `global.random_seed` | Reproducibility seed (integer) |
| `global.output_format` | `csv` or `parquet` |
| `temporal.start_date / end_date` | Date range for generated timestamps |
| `entities.{name}.count` | Row count per entity |
| `data_quality.missing_rates` | NULL injection rates per field |
| `data_quality.duplicate_rates` | Near-duplicate injection rates |
| `data_quality.anomaly_rates` | Sensor anomaly injection rates |
| `sensor_ranges` | Normal and anomalous operating ranges per sensor type |

## Generation Order (Dependency)

Entities are always generated in dependency order to maintain FK integrity:

```
production_lines → shifts → defect_types → machines → employees → products
    → production_orders → downtime_events → sensor_readings
    → quality_inspections → pm_schedules → maintenance_logs → material_movements
```

The `Registry` class maintains in-memory lists of all generated PKs,
enabling child generators to sample valid foreign keys deterministically.

---

## Data Quality Features

| Feature | Implementation |
|---|---|
| Missing values | `AnomalyInjector.maybe_null()` with configurable per-field rates |
| Sensor anomalies | `AnomalyInjector.sensor_value()` with out-of-range value generation |
| Near-duplicates | `AnomalyInjector.inject_duplicates()` with modified PK suffix |
| Data quality scores | `AnomalyInjector.data_quality_score()` with beta distribution |
| Determinism | All `random.Random` instances seeded from global seed + class hash |

These CSV files can be loaded directly into the SMAP operational database
using PostgreSQL `COPY` commands or the ETL pipeline's seed loader.
